MathModule                                                        *MathModule*


    Fields: ~
        {int_to_hex_string} (function)  convert an integer colour to a
                                        "#rrggbb" string
        {rgb}               (function)  split a "#rrggbb" string into {r, g,
                                        b} channels
        {blend}             (function)  mix two colours by an amount from 0
                                        (bg) to 1 (fg)


M.int_to_hex_string()                                  *M.int_to_hex_string()*
    Will turn a integer colour value into a string hex value




================================================================================
ConfigModule                                                    *ConfigModule*


    Fields: ~
        {defaults} (Config)  default options
        {options}  (Config)  config table extending defaults


Highlight                                                          *Highlight*


    Fields: ~
        {fg} (string)  foreground colour, "#rrggbb"
        {bg} (string)  background colour, "#rrggbb"


Extend                                                                *Extend*


    Fields: ~
        {line_number} (boolean)  extend the cursorline colour into the
                                 line-number column
        {signs}       (boolean)  extend the cursorline colour into the sign
                                 column
        {folds}       (boolean)  extend the cursorline colour into the fold
                                 column


Disabled                                                            *Disabled*


    Fields: ~
        {filetypes} (string[])  filetypes moody leaves alone (e.g.
                                "TelescopePrompt")
        {buftypes}  (string[])  buftypes moody leaves alone (e.g. "nofile")


Recording                                                          *Recording*


    Fields: ~
        {enabled}       (boolean)  show a macro-recording indicator at the end
                                   of the cursorline
        {icon}          (string)   icon shown next to the recording register
        {prefix}        (string)   text shown before the register character
        {suffix}        (string)   text shown after the register character
        {right_padding} (integer)  cells of right padding (shifts the
                                   indicator left)


ColumnSeparator                                              *ColumnSeparator*


    Fields: ~
        {char}      (string)     character drawn between the moody column and
                                 the code
        {highlight} (Highlight)  separator highlight; defaults to the
                                 CursorLine background


ColumnFolds                                                      *ColumnFolds*


    Fields: ~
        {enabled}     (boolean)  render folds in the moody column
        {start_color} (string)   gradient start colour for fold levels,
                                 "#rrggbb"
        {end_color}   (string)   gradient end colour for fold levels,
                                 "#rrggbb"


ColumnMarks                                                      *ColumnMarks*


    Fields: ~
        {enabled}    (boolean)  render marks in the moody column
        {alphabetic} (boolean)  show alphabetic (a-z) marks
        {other}      (boolean)  show non-alphabetic marks


Column                                                                *Column*


    Fields: ~
        {enabled}   (boolean)          replace the statuscolumn with moody's
                                       own
        {numbers}   (boolean)          render line numbers
        {signs}     (boolean)          render signs
        {folds}     (ColumnFolds)      fold rendering + gradient
        {marks}     (ColumnMarks)      mark rendering
        {highlight} (Highlight)        base highlight for the column
                                       (typically just a bg)
        {separator} (ColumnSeparator)  separator between the column and the
                                       code


Config                                                                *Config*


    Fields: ~
        {colors}           (Colors)         per-mode base colour (a `*Moody`
                                            highlight group overrides it)
        {blends}           (Blends|number)  per-mode blend amount toward the
                                            background, or one number for all
        {bold_line_number} (boolean)        bold the cursor-line number
        {line_number_only} (boolean)        colour only the line number,
                                            leaving the default cursorline
        {extend}           (Extend)         extend the cursorline colour into
                                            adjacent columns
        {disabled}         (Disabled)       filetypes/buftypes to leave alone
        {recording}        (Recording)      macro-recording indicator
        {column}           (Column)         moody's own statuscolumn


M.is_disabled()                                              *M.is_disabled()*


    Return: ~
        boolean: true if filetype was in list of disabled filetypes or
        buftypes


M.defaults()                                                    *M.defaults()*
